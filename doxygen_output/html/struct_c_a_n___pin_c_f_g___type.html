uint8_t RD

Serial Inputs, from CAN transceivers, should be: For CAN1:

  • CAN_RD1_P0_0: RD pin is on P0.0
  • CAN_RD1_P0_21 : RD pin is on P0.21 For CAN2:
  • CAN_RD2_P0_4: RD pin is on P0.4
  • CAN_RD2_P2_7: RD pin is on P2.7
